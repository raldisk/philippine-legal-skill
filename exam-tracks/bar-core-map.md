# Bar Core Map — Philippine Bar Examination

## Purpose and Constraint

**Overlay only.** This file contains no doctrine. It maps Philippine Bar Examination subjects to existing skill modules. For all substantive analysis, load the routed module. Do not derive legal conclusions from this file.

**Exam authority**: Supreme Court of the Philippines, Bar Examination Committee. Subject weights and coverage areas below are drawn from the 2011 Bar Exam reforms and the 2024 Bar Exam guidelines. Verify the current year's SC-released coverage checklist for any adjustments.
